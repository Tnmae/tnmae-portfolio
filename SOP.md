# SOP — Developer Portfolio Website

Build an immersive, scroll-driven single-page portfolio inspired by Google Arts & Culture's Meroë experience. The site uses full-viewport sections, cinematic background visuals, parallax transitions, and smooth anchor navigation to guide recruiters through a narrative: Hero → About → Skills → Projects → Contact. Each section reveals on scroll with staggered animations. Dark-themed, typographically bold, minimal UI chrome. Deploys as a static site (HTML/CSS/JS or React + Vite).
